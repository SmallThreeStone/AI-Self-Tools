# 自用工具箱 (Personal Toolbox)

> 微信小程序 · 本地工具聚合 · 数据仅存本机

一个离线优先的微信工具箱小程序，集成 30+ 个实用小工具，涵盖效率、生活、社交、创意、健身、学单词等场景。所有数据存储在本地，无需网络即可使用。

---

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | 微信原生小程序 + TypeScript |
| 基础库 | libVersion 3.5.5+ |
| 二维码 | uqrcodejs |
| 样式 | WXSS + CSS 变量（明暗主题） |
| 存储 | wx.Storage（纯本地，无后端/无云服务） |

---

## 快速开始

```bash
# 1. 克隆仓库
git clone git@github.com:SmallThreeStone/AI-Self-Tools.git

# 2. 安装依赖（仅 TypeScript 类型 + uqrcodejs）
npm install

# 3. 用微信开发者工具打开项目根目录
#    - 在 project.config.json 中填入你的 AppID
#    - 或使用测试号直接预览
```

> **注意**：`project.private.config.json` 由开发者工具自动生成，不要提交到仓库。

## 核心特性

### 离线优先
- 所有数据通过 `wx.Storage` 存储在本机
- 无需后端服务，无需网络
- 工具注册表配置文件驱动

### 数据备份
- 完整备份 JSON（v2 格式）含收藏/统计/工具数据
- 支持合并/覆盖两种导入模式
- 收藏列表可单独导出/导入
- 支持从聊天文件选取 JSON 恢复

### 外观
- 三态主题：跟随系统 / 浅色 / 深色
- 系统主题变化实时同步
- CSS 变量全局控制

### 分享
- 工具页分享带参数，承接方自动打开对应工具

## 开发

```bash
# TypeScript 编译检查
npx tsc --noEmit

# 在微信开发者工具中打开项目根目录即可编译运行
```

### 添加新工具

1. 在 `miniprogram/config/tools.ts` 的 `tools` 数组中注册
2. 在对应分包目录创建页面文件
3. 在 `miniprogram/app.json` 的 `subpackages[].pages` 中添加路径

```typescript
// 示例：注册新工具
{
  id: 'my-tool',
  name: '我的工具',
  desc: '一句话描述',
  categoryId: 'efficiency',
  path: '/package-tools/my-tool/index',
  icon: '🆕',
  tags: ['标签1', '标签2'],
}
```
