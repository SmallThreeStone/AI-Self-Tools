# 工具箱小程序 · 产品规划 v1.4.x 总册（学单词模块）

> **文档索引**：[../README.md](../README.md)

> **前置文档**：[产品规划-v1.3.0.md](./产品规划-v1.3.0.md) · [产品规划-v1.2.0.md](./产品规划-v1.2.0.md)

## 一、愿景与交付顺序（优先级从高到低）

| 顺序 | 文档 | 主题 | 优先级 | 说明 |
|------|------|------|--------|------|
| 1 | **[产品规划-v1.4.1.md](./产品规划-v1.4.1.md)** | **单词本** | **P0 先交付** | 日常 / 外贸分栏、中英搜索、词条详情（例句 + 词/句读音）；可 **尽量本地化** 或仅 TTS 走云。 |
| 2 | **[产品规划-v1.4.2.md](./产品规划-v1.4.2.md)** | **文字对话** | **P1** | 大模型 **文字** 对话、与词本弱联动；验证成本与合规，**无录音**。 |
| 3 | **[产品规划-v1.4.3.md](./产品规划-v1.4.3.md)** | **类豆包语音对话** | **P2 靠后** | 录音 + ASR + LLM + TTS 全链路；依赖 **§合规** 与厂商选型，建议前两册稳定后再立项。 |

**产品一句话**：先 **「查得到、听得懂、有例句」**，再 **「问得到」**，最后 **「像豆包一样说」**。

## 二、模块结构（总览 IA）

```
学单词（入口：建议首页 + 分包 package-learn，见 v1.4.1）
├── 单词本（v1.4.1）—— 日常 | 外贸、搜索、详情
├── 对话·文字（v1.4.2）
└── 对话·语音（v1.4.3）
```

## 三、跨子文档的共性要求

| 项 | 说明 |
|------|------|
| **分包** | 建议统一 **`package-learn`**（或 `package-words`），子功能分页面即可。 |
| **登录** | 语音对话 **强烈建议登录后可用**（v1.4.3）；单词本是否游客可读 **在 v1.4.1 §开放问题** 闭合。 |
| **合规** | 生成式与深度合成以 **v1.4.2 / v1.4.3** 为主；词表版权以 **v1.4.1** 为主。 |
