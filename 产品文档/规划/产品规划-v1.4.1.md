# 工具箱小程序 · 产品规划（v1.4.1 · 单词本）

> **总册索引**：[产品规划-v1.4.0.md](./产品规划-v1.4.0.md)  
> **文档索引**：[../README.md](../README.md)  
> **优先级**：**P0（学单词大线中优先交付）**  
> **依赖**：可与 v1.4.2 / v1.4.3 **解耦**；无对话亦可独立发版。

---

## 一、目标

交付 **「双分类单词本 + 中英搜索 + 词条详情（例句 + 读音）」**，满足日常学习与外贸术语两类场景；**不要求**本期上线文字或语音对话（无结果文案可不引用「问 AI」）。

---

## 二、用户与场景

| 人群 | 场景 | 本期要满足的诉求 |
|------|------|-------------------|
| 日常学习者 | 碎片查词、记义 | 快搜、例句、能听 **单词 + 例句** |
| 外贸 / 业务人员 | 报价、邮件前查术语 | **独立外贸词栏**、例句偏商务、可检索 |

---

## 三、信息架构

| 页面 / 区块 | 说明 |
|-------------|------|
| **学单词首页** | 入口：单词本为主；顶部或首屏 **搜索框**（占位提示：「中 / 英文搜索」）。 |
| **单词本** | 顶部 **Segmented / Tab**：**「日常单词」|「外贸词汇」**；下方列表（可分页或虚拟列表）。 |
| **搜索结果** | 列表展示匹配词头 + 简要释义；点击进入 **词条详情**。 |
| **词条详情** | 见 §五。 |

---

## 四、分类与词表策略

| 分类 | 定位 | 数据 |
|------|------|------|
| **日常单词** | 通用高频、生活 / 考试向 | 独立词表或 `category=daily`；版权可商用。 |
| **外贸词汇** | 单证、物流、支付、谈判等 | 独立词表或 `category=trade`；**与日常字段一致**，避免合并后丢失标签。 |

**排序（P1）**：按字母 / 频率 / 最近更新。首期可仅字母序。

---

## 五、搜索

| 需求 | 验收要点 |
|------|----------|
| **英文搜** | `headword` 前缀 / 子串；**大小写不敏感**；输入防抖约 **300ms**。 |
| **中文搜** | 搜 **中文释义**、必要时近义标签；无结果时 **友好提示** + 清空建议（本期不要求跳转 AI）。 |
| **高亮** | 列表与详情中 **匹配片段高亮**。 |

**性能**：词量较大时评估 **本地索引** 或 **服务端搜索接口**（研发选型）。

---

## 六、词条详情（核心）

| 区块 | 内容 | 验收 |
|------|------|------|
| **词头** | 英文拼写 + **音标**（IPA/KK，与词库一致）+ **主中文释义** | 缺音标时 UI 不留白块，可隐藏行 |
| **单词读音** | **播放** 按钮：TTS 或预置音频 | 可重复点击重播；弱网有提示 |
| **例句** | **至少 1 条**，语境与 **当前 Tab（日常/外贸）** 一致 | 版权清晰，忌低质机翻堆砌 |
| **例句读音** | **整句播放** 按钮 | **与单词播放分离**；用户能区分「词怎么读、句怎么读」 |

**最低标准**：**词、句各一条可播放读音**（TTS 或静态音频二选一或并存）。

**P1 增强**：例句下弱标注（分词重音）、例句复制、收藏进「生词本」。

---

## 七、数据与工程

| 项 | 建议 |
|------|------|
| **分包** | `package-learn`，词表 **JSON / SQLite** 放分包。 |
| **字段示例** | `id`, `headword`, `phonetic`, `zh`, `category`（`daily` \| `trade`）, `examples[]`（`text`, `zh_hint?`, `audio_sentence_url?`）, `audio_word_url?`, `tags[]` |
| **TTS** | 预生成 URL 或实时 TTS API；统一 **音色、语速**；做 **失败降级**（仅展示文本）。 |
| **主题** | 与全局明暗主题一致（如 `applyThemeForToolPage`）。 |

---

## 八、成功指标

| 指标 | 说明 |
|------|------|
| 搜索有结果占比 / 无结果退出率 | 衡量词表覆盖与检索质量 |
| 详情页 **单词播放 / 例句播放** 点击率 | 衡量「读音」价值是否被感知 |

---

## 九、风险

| 风险 | 应对 |
|------|------|
| 词表版权 | 仅用授权数据；外贸词优先 **采购或自建** |
| 分包体积 | 控制首包词量或按需下载扩展包（P2） |

---

## 十、开放问题

1. 单词本是否 **允许游客只读**（不落账号）？  
2. 首期 **日常 / 外贸各多少词**、更新节奏？  
3. 「怎么读」是否 **必须展示音标**（无音标仅 TTS 是否接受）？
